* TITLE Command
This example is for multiple-group path analysis conducted in Mplus.
#+NAME: TITLE
#+BEGIN_SRC inp :tangle yes
TITLE:
  Example 4 - Multiple Linear Regression - 02-2 Multiple-Group Path Analysis
#+END_SRC

* DATA Command
Nothing changes with the data importation. 
#+NAME: DATA
#+BEGIN_SRC inp :tangle yes
DATA:
  FILE IS "../../data/job_placement.csv";
#+END_SRC

* VARIABLE Command
The data are imported the same way as before, but notice the change
under the USEVARIABLES ARE statement. The GOUPRING IS command
specifies the grouping varibale for the path model - a path model will
be estimated separately for the male and female group in this example.
#+NAME: VARIABLE
#+BEGIN_SRC inp :tangle yes
VARIABLE:
  NAMES ARE
  id wjcalc wjspl wratspl wratcalc waiscalc waisspl
  edlevel newschl suspend expelled haveld female age;

  USEVARIABLES ARE
  wjspl edlevel age;
  
  MISSING ARE all(99999);
  
  GROUPING IS female(0 = male, 1 = female);
#+END_SRC

* ANALYSIS Command
Bootstrapped standard errors can be computed using the BOOTSTRAP
command and the number of bootstrapped draws can be specified here as
well.
#+NAME: ANALYSIS
#+BEGIN_SRC inp :tangle yes
ANALYSIS:
  BOOTSTRAP = 1000;
#+END_SRC

* MODEL Command
The path model (or indirect-effect, or mediation model) being
specified here has three variables: wjspl (the outcome), age (the
predictor), and edlevel (the mediator). The hypothesis being tested is
that the influence of age on wjspl can be explained (or partially
explained) by edlevel. Under the MODEL command, three direct
regressive paths are specified with the ON command. The MODEL INDIRECT
command specifies an indirect regressive path from age to wjspl
through edlevel.
#+NAME: MODEL
#+BEGIN_SRC inp :tangle yes
MODEL:
  wjspl ON edlevel age;
  edlevel ON age;
  
MODEL INDIRECT:
  wjspl IND edlevel age;
#+END_SRC

* OUTPUT Command
The CINTERVAL(BOOTSTRAP) is used to request the confidence intervals
based on the bootstrapped standard errors.
#+NAME: OUTPUT
#+BEGIN_SRC inp :tangle yes
OUTPUT:
  TECH1 STANDARDIZED CINTERVAL(BOOTSTRAP);
#+END_SRC

	mplus reg-02-2.inp
